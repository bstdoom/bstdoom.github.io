# Mine the archived vocal theme for reusable layout and CSS patterns

The `.todo/theme/` folder contains the original purchased vocal theme. Keep it as a reference asset, not active site code.

## Goal

- Audit the theme for reusable layout and CSS ideas for future pages like concerts, gallery, or other non-content pages.
- Extract any useful spacing, color, typography, and structural patterns into the Kobweb site when needed.

## Suggested follow-up work

- Inventory the theme's reusable pieces.
- Note which files are still relevant and which are legacy-only.
- Decide whether any layout fragments should be ported into active site CSS/components.
