Current contents:

- `theme/`
  - the original purchased vocal theme
  - keep for future pages and layout/CSS reference
- `grav-source.html`
  - static dump of the final Grav home page
  - keep as a compact reference for imports, structure, and content details
- `reviews/`
  - canonical review sources and legacy sidebar imports
- `images/`
  - preserved image assets from the old site and local archives
- `screenshots/`
  - theme and site screenshots that are useful as visual references

When we start follow-up cleanup tasks, this folder is the place to mine for source references before opening issues or moving anything into the active site code.
